Qubit Sending Problem
Goal: $P_1$ wants to send a qubit to $P_2$.
We limit the adversary to:
Drop outgoing messages.
Claim message wasn't sent.
Our solution: Routing
Packets: Quantum error-correcting code
Termination condition:
$P_1$ has no packets left, or
no path exists from $P_1$ to $P_2$.
Bound on packet loss: number of edges.
We can identify the cheater!
Graph is disconnected.
Edges never break between honest parties.
$\implies$ Honest parties are on the same connected component.
General adversary behavior $\rightarrow$ use authentication code.
